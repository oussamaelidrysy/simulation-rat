## Lancement de l'application

### Installation des bibliothèques nécessaires
Exécuter la commande suivante dans un terminal ouvert dans le dossier contenant ce fichier, le fichier requirements.txt et les fichiers de code app.py et main.py.
```bash
pip install -r requirements.txt
```

### Lancement de l'application
Pour lancer l'interface, exécuter la commande suivante dans le même terminal. L'interface s'ouvrira automatiquement dans votre navigateur par défaut.
```bash
streamlit run app.py
```
